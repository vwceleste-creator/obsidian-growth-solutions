# Obsidian Growth Solutions Website

Static GitHub-ready website.

## Files
- `index.html`
- `style.css`
- `script.js`
- `/assets` brand files and images

## GitHub Pages deploy
1. Upload all files to your repository root.
2. Go to Settings → Pages.
3. Source: Deploy from branch.
4. Branch: `main`, folder: `/root`.
5. Save.

## Domain
Use `obsidiansolutions.co.za` once DNS is pointed to the chosen host.
